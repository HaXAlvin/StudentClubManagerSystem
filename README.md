# 學生社團管理系統
本專案是基於Python Flask所架構的Web Server
使用到JWT令牌驗證、SHA512雜揍加密、QRCode打卡、APScheduler等技術
主要目的是提供學生社團更好的平台，功能包括社課打卡、請假、設備借用、活動報名、公告、後台數據、後台核假等功能 

本專案參加108學年度逢甲大學資訊工程學系深碗專題競賽  
榮獲 最佳人氣獎 與 優秀作品獎
